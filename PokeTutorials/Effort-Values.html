What are Effort Values?

Effort Values are hidden stats that a Pokémon can gain by catching and defeating Pokémon. Every Species has 0 to 3 Effort Points for each main stat it gives to the enemy on defeat/catch. To read more on Effort Values I would like to refer to Bulbapedia

From now on in the tutorial Effort Values will be called EV’s.

GlobalSettings.cs

The first thing we are going to update are the GlobalSettings. If you haven’t followed any of my tutorials before u might learn quickly I really enjoy bringing constants to the GlobalSettings script. This also helps a lot with testing by not hardcoding them.

We need to add 2 new values there. The first 1 is the max EV’s a pokemon can get in total. And the other is the max a Pokémon can get in a single stat. I also added a new header for the inspector and getters to get the new values.


[Header("Pokémon")]
[SerializeField] int maxEvs = 510;
[SerializeField] int maxEvPerStat = 252;

public int MaxEvs => maxEvs;
public int MaxEvPerStat => maxEvPerStat;

Next we are going to turn these values into a Dictionary. The reason I’m using a dictionary is to loop through the Ev’s easily. We only have 1 issue. We never had any value in Stat for HitPoints (HP). To fix this without breaking the existing Enum we just add it at the end of the Stat Enum. It will look like this from now on: 

public enum Stat
{
	Attack,
	Defense,
	SpAttack,
	SpDefense,
	Speed,

	// These 2 are not actual stats, they're used to boost the moveAccuracy
	Accuracy,
	Evasion,

	// Added HP for EV's
	HitPoints
}
Now we have the Enum, we can finish our last step in the PokemonBase. Which is the Getter: 

public Dictionary EvYields => new Dictionary() {
    	{ Stat.HitPoints, hitPointsEvYield },
    	{ Stat.Attack, attackEvYield },
    	{ Stat.Defense, defenseEvYield },
    	{ Stat.SpAttack, spAttackEvYield },
    	{ Stat.SpDefense, spDefenseEvYield },
    	{ Stat.Speed, speedEvYield }
};

Pokemon.cs

Next up we are actually updating the Pokémon themself. First we add the Dictionary that will hold the EV’s our Pokémon has earned. 


public Dictionary StatEffortValues { get; private set; }
Next we update the Init() method to set the StatEffortValues Dictionary to its default values. This we do right before we set our EXP and right after we set our Moves. 

StatEffortValues = new Dictionary() { { Stat.HitPoints, 0 }, { Stat.Attack, 0 }, { Stat.Defense, 0 }, { Stat.SpAttack, 0 }, { Stat.SpDefense, 0 }, { Stat.Speed, 0 } };
Next we need 2 new Methods. The first method is 1 we use to gain EV’s and the second is a helper method to calculate the max EV’s we currently have. 

public void GainEvs(Dictionary evGained)
{
  // sev stands for "Stat Effort Values"
  foreach (var sev in StatEffortValues.ToArray())
  {
    if (sev.Value < GlobalSettings.i.MaxEvPerStat && GetTotalEvs() < GlobalSettings.i.MaxEvs)
    {
      evGained[sev.Key] = Mathf.Clamp(evGained[sev.Key], 0, (GlobalSettings.i.MaxEvs - GetTotalEvs()));
      StatEffortValues[sev.Key] = Mathf.Clamp((StatEffortValues[sev.Key] += evGained[sev.Key]), 0, GlobalSettings.i.MaxEvPerStat);
    }
  }
}

public int GetTotalEvs()
{
  return StatEffortValues.Values.Sum();
}

void CalculateStats()

    	Stats.Add(Stat.Attack, Mathf.FloorToInt((((2f * Base.Attack + (StatEffortValues[Stat.Attack] / 4f)) * Level) / 100f) + 5f)); // Todo Nature + IV's

    	Stats.Add(Stat.SpAttack, Mathf.FloorToInt((((2f * Base.SpAttack + (StatEffortValues[Stat.SpAttack] / 4f)) * Level) / 100f) + 5f));  // Todo Nature + IV's

    	Stats.Add(Stat.Speed, Mathf.FloorToInt((((2f * Base.Speed + (StatEffortValues[Stat.Speed] / 4f)) * Level) / 100f) + 5f));  // Todo Nature + IV's

    	int oldMaxHP = MaxHp;
    	MaxHp = Mathf.FloorToInt((((2f * Base.MaxHp + (StatEffortValues[Stat.HitPoints] / 4f)) * Level) / 100f) + Level + 10f); // Todo IV's

    	if (oldMaxHP != 0)
        	HP += MaxHp - oldMaxHP;
